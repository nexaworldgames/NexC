
# NexC

## ⬇️ Download the Latest Release

Get the newest stable NexC version from the **Latest Release** before using the project.

---

NexC is a lightweight, Lua-inspired scripting language developed under **NexaWorldGames**. It is designed to be simple, fast, and practical, while being powerful enough to integrate with real systems.

## 🚀 About NexaWorldGames

NexaWorldGames is focused on building modern infrastructure, automation systems, and hosting solutions. NexC is a core part of this ecosystem — created to power scripts, automation, and future tools.

---

## ⚡ Vision

NexC is built with a clear direction:

* Simple and readable syntax
* Fast and lightweight execution
* Minimal setup required
* Designed for real-world use

The goal is to create a scripting language that is easy to use, but strong enough to scale with growing systems.

---

## 🔥 Features

* Clean and intuitive syntax
* Variables and expressions
* Conditional logic (`if`, `else`, `elseif`)
* Loops (`while`, future `for` support)
* Functions and return values
* Lists and maps
* Built-in utilities (strings, arrays, files)
* Error handling (`try`, `catch`)

---

## 📦 Example

```nexc
let name = "NexaWorldGames"
let score = 5

print("hello from NexC")

if score > 3 then
    print("score is good")
else
    print("score is low")
end

function greet(user)
    print("hi " + user)
end

greet(name)
```

---

## ▶️ Running NexC

Currently, NexC runs through its interpreter:

```bash
py nexc.py script.nxc
```

In the future, NexC will run as a standalone tool:

```bash
nexc script.nxc
```
