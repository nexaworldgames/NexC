# NexC v1.0

NexC is a small Lua-like scripting language prototype.

## Run
Use Python 3:

```bash
python nexc.py examples/hello.nxc
```

## Features in this first version
- `let` / assignment
- `print(...)`
- `if / elseif / else / end`
- `while ... end`
- `function ... end`
- `return`
- arrays and maps
- basic built-ins for math, strings, and files
- simple error handling with `try ... catch ... end`

## Notes
This is the first usable version. The 250 words are included as reserved words / vocabulary for the language roadmap, but not every one is fully implemented yet.
